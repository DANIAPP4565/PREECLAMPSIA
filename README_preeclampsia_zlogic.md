# App Preeclampsia ICG Z-Logic

Versión 2026-06-14k.

## Cambio crítico real

La pestaña **PDF Z-Logic: dos informes** tiene dos importadores separados y dos botones independientes:

1. **PDF COMPLETO Z-Logic**  
   Botón: **1. IMPORTAR INFORME COMPLETO Z-LOGIC**

2. **PDF Z-Logic DE 4 HOJAS**  
   Botón: **2. IMPORTAR INFORME DE 4 HOJAS**

3. Botón final: **3. COMPLEMENTAR VARIABLES Y ANALIZAR COMPLETO + 4 HOJAS**

El informe completo y el informe de 4 hojas se integran para complementar variables. El informe de 4 hojas aporta ortostatismo acostado/sentado/parado y variables adicionales como CFTnr, EA, EES, AC.

## Ejecución

```bash
pip install -r requirements.txt
streamlit run app.py
```
